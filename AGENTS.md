This is my fork of [eclipse.jdt.ls](https://github.com/eclipse-jdtls/eclipse.jdt.ls), a Java language server.

I keep a fork to be able to experiment on my own, but I would of course prefer any improvements to be eventually merged upstream.

However, for the time being, I want to optimize for smootly being able to merge in changes from upstream with a minimum of merge conflicts. Therefore, changes should try to be as additive-only as possible and wherever possible, alter existing code by adding clean seams.
